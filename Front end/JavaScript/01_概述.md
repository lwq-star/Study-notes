## 概述

- **JavaScript** ：是一种弱类型脚本语言，其源代码不需要经过编译，而是由浏览器解释运行

- **ECMAScript** : 可以理解为JavaScript的一个标准



### 1.快速入门

#### 1.1、引入Javascript

1. 内部标签

    ```html
    <!--script标签内，写Javascript代码-->
    <!--内部样式-->
    <script>
        // 弹窗
        alert("hello, world");
    </script>
    ```

2. 外部引入

    mj.js

    ```javascript
    alert("hello, world");
    ```

    test.html

    ```html
    <!--外部引入-->
    <!--注意：script：必须成对出现-->
    <script src="mj.js"></script>
    ```
