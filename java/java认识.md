## JDK

java开发者工具, java和javac用来编译运行java，javadoc将java的某些东西生成一个文档，jar将java包装为一个应用

## JRE

JDK包含了JRE，JRE为java的运行环境，只要有JRE就能运行java程序

## JVM

java虚拟机，用于跨平台
